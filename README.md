# 上海家庭活动雷达

把分散在十几个平台的**上海体育赛事 / 行业展会 / 文化演出**聚合到一个微信小程序，
按「亲子 / 区域 / 时间 / 价格」筛选，支持收藏与开票提醒。家人各自浏览订阅，爸爸在海外每周收精选。

- 形态：微信小程序（个人号起步）+ 微信云开发 + 本机 Python 采集
- 定位：信息发现 + 提醒，不卖票、跳官方渠道
- 状态：**M1 采集层已跑通**（抓到真实数据），云/前端骨架已就绪

## 快速开始

```bash
# 1. 采集层（立即可验证）
cd collector && pip3 install -r requirements.txt && python3 main.py
#   → 产物 data/events.json

# 2. 云后端：见 cloud/README.md（开通云开发、建集合、部署函数）
# 3. 小程序：微信开发者工具导入 miniprogram/，见 miniprogram/README.md
```

## 需要你提供 / 操作

1. 微信小程序 **AppId**（注册个人号即可）
2. **文化上海云抓包接口**（在国内浏览器抓，见 `docs/数据源对接清单.md`）
3. 开通**微信云开发环境**并填环境 ID
